## 💻 Getting Started <a name="getting-started"></a>

To get a local copy.

### Prerequisites

In order to run this project you need to have Ruby and react/redux installed, check [RoR](https://rubyonrails.org/) and [React/Redux](https://react-redux.js.org/) documentation in order to know how to install it.

### Setup

Clone this repository to your desired folder:

sh<br />
cd my-folder<br />
git clone https://github.com/Dachrono/hello-rails-react.git 

### Install

You can install the gems with:

```sh
  bundle install
```

You can install the npm packages with:

```sh
  npm install
```

### Usage

To run the project, execute the command below in your terminal:

```sh
   ./bin/dev
```

Open you browser and go to http://localhost:3000/
